Проект "Отток клиентов банка"

Из «Бета-Банка» стали уходить клиенты. Каждый месяц. Немного, но заметно. Банковские маркетологи посчитали: сохранять текущих клиентов дешевле, чем привлекать новых.

Цель проекта

Необходимо спрогнозировать, уйдёт клиент из банка в ближайшее время или нет. 
Представлены исторические данные о поведении клиентов и расторжении договоров с банком.
Необходимо построить модель с предельно большим значением F1-меры. Нужно довести метрику до 0.59. Далее нужно проверит F1-меру на тестовой выборке.

Итого исследования

Для выполнения поcтавленной задачи (доведение метрики F1-меры) до 0,59 были проверены три модели DecisionTree, RandomForest, LogisticRegression.
В процессе обучения моделей на валидационной выборке, лучшие результаты показала модель RandomForest.
Данную модель мы и использовали на тестовой выборке, на которой модель показала результат F1 равный 0,62.
